# Forked-Paper

This repository is for maintaining the Minecraft server software used in OKOCRAFT. 
We currently maintain a hard forked version from Pufferfish.

## Building

```bash
./gradlew build
```

Or building a Paperclip JAR for distribution:

```bash
./gradlew paperclip
```
